Frequently Asked Questions (FAQs)

What is the purpose of this website?

I created this website to display it in my portfolio. I grabbed my inspiration from 2 things that I love the most in life, which are physics and coding. I used interesting videos about science that are easy to watch and showcased them on this website.

Which technologies did you use to create this website?

I used HTML5, CSS3 and JavaScript. I also used SASS to organize and reuse my CSS code and all the progress that I made was uploaded to my GitHub page

Where do the science videos come from?

All videos used in this website are embedded from YouTube, and all credits for the videos go to their respective creators.

Why can't I log in or sign up to the website?

This website is not backed by a database nor a backend (because it lives in GitHub Pages). That is why those options are not available right now.

Where can I see the code of this website?

You can see all the website's code on my GitHub repository called "The Scientific Room".

How can I contact you?

You can write me via E-Mail (estebansant30@gmail.com) or send me a message through LinkedIn or Twitter. I will surely be answering you shortly.
